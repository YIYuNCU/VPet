## 应用作用

修改数据文件

## 使用方式

1.开启数据库

1. 找到数据文件(mod\9011_BiliTest\Data\LikeContentDatabase.sqlite)，鼠标右键点击，复制地址

2. 找到DataBaseControl.exe，双击打开

3. 用复制的地址替换“数据库文件路径”

4. 点击“开启数据库”，关闭两个提示窗(使用mod内按键，不需要执行1-3)

2.修改数据

1. 双击要修改的数据，填入合适的数据类型的数据

2. 点击Update按键

3.新增数据(大概率不需要)

1. 在中间的一行输入框中依次填写符合数据类型的数据，输入框内容与下方数据的标头一一对应，  适合的数据类型为(字符串，字符串，任意整数，任意数，任意整数，任意整数，任意数，0-3的数字，任意整数)，全部填写完成后敲击空格

2. 单击Add按键

4.删除数据

1. 单击要删除的数据所在行

2. 单击"Delete"按键
